# 判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

```
var isPalindrome = function(x) {
    x = x.toString()
    var len = Math.floor(x.length / 2)
    var arr = x.split("")
    let tag = true
    for(var i = 0;i<len;i++){
        if(arr[i]!==arr[arr.length -1-i]){
            tag = false
        }
    }
    return tag
};
```
